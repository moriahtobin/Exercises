Notes of Interest

There are two line breaks <p>,</p> and <br>. The first allows the text to be separated (maybe by a picture? we'll see!) and the second keeps the text together as one unit like a stanza in a poem.

id's can be added to tags so that you can reference them on other parts of the page. This is the syntax for page teleportations #intraSiteTeleportation

Hilarious aside to my personal history, it seems that the DOMs that I am used to are not the DOMs of web developers. DOMs are the downward-facing PMTs (light detectors) that record the particle interactions on the IceCube Neutrino Observatory (where I did my graduate work).

So it turns out there are two "cite"s": <blockquote cite = "URL"> which is a hidden cite attribute seen mainly by search engines behind the scenes, and <p>I just learned something sweet in <cite>Book Title</cite></p>, the cite tag which is used to reference the title of a creative work.

<pre> seems like an essential tag for python code, since it keeps track of white space of the text inside.
